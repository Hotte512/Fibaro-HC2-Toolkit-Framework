HC2-Toolkit-Framework

This Framework is a lua library for HC2 and is an addon for HC2 Toolkit application in a goal to aid the integration. Hope that it will be useful!

Current version 1.0.2 [12-13-2013]

Tested on Lua 5.1 with Fibaro HC2 3.5xx

Informations: Memory is preserved: The code is loaded only the first time in a virtual device main loop and reloaded only if virtual device is "saved" or application pool restarded.
